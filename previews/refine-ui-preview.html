UI 方向性プレビュー v2.2 — グラスモーフィズム / パステル / リッチアニメ
このプレビューでわかること:
  • グラス化: ステータスバー / スキルゲージ / 盤面コンテナ — 背景の色が透けて見える
  • パステル化: 犬・猫マスとセル背景。数字 (1, 2/1 …) は視認性のため彩度を残している

  • スキルゲージ100%演出: 斜め白シーン / 猫アイコンのグロー強弱 / カラーグラデーション
  • スコアブレイクダウン: 行が左から 40ms ずつカスケードイン
ステージ切り替えで背景色が変わり、それに応じてグラスの透け方も変化します。
