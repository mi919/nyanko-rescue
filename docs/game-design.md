## 1. ゲームコンセプト

### 1-1. ワンライン

「マインスイーパーに猫保護と仲間集めを融合した、温かみのあるパズルゲーム」

### 1-2. コアループ

```
ステージ開始 → マスを開く → 数字を頼りに猫を探す → 犬を避ける
  → 全猫保護でクリア → 次ステージへ → はいきょクリアで1周完了
  → 仲間との出会い → 新しいお供をゲット → より強いスキルで再挑戦
```

### 1-3. プレイヤー体験の柱

| 柱 | 内容 |
|----|------|
| **探索の緊張感** | 犬か猫か分からないマスを開けるドキドキ |
| **保護の喜び** | 猫を見つけたときのキラキラ演出と達成感 |
| **成長の実感** | 周回ごとに新しい仲間が増え、選択肢が広がる |
| **戦略の深み** | スキルの選択と発動タイミングで攻略法が変わる |

### 1-4. マインスイーパーとの差別化

| 要素 | マインスイーパー | にゃんこレスキュー |
|------|----------------|------------------|
| 目的 | 地雷を避けて全安全マスを開く | 猫を見つけて全匹保護する |
| 危険物 | 地雷（即死） | 野良犬（ライフ制、3回まで許容） |
| 数字ヒント | 周囲の地雷数（1種類） | 犬の数＋猫の数（赤・緑の2色で表示） |
| 終了条件 | 全安全マス開封 | 全猫保護（安全マス開封は不要） |
| 収集要素 | なし | 猫コレクション図鑑（10種） |
| スキル | なし | お供の猫による10種類のスキル |
| 周回要素 | なし | 難易度上昇 + 仲間アンロック |

## 2. ゲームルール

### 2-1. 基本ルール

1. グリッド上の**未開封マス**をタップして開ける
2. 開けたマスには周囲8マスの動物情報が表示される
   - **赤い数字**: 周囲の野良犬の数（0〜8）
   - **緑の数字**: 周囲の猫の数（0〜8）
   - 表示形式: `犬数 / 猫数`（例: `2 / 1`）
3. 周囲に犬も猫もいない空マス（0/0）を開けると、**フラッドフィル**で隣接安全マスが連鎖的に開く
4. **猫マスを開ける** → 保護成功（スコア・ゲージ加算）
5. **犬マスを開ける** → 噛みつき演出 → ライフ-1
6. 全猫保護 → **ステージクリア**
7. ライフ0 → **ゲームオーバー**

### 2-2. フラグモード

モバイル環境での長押し操作の信頼性問題を解決するため、**トグルボタン方式**を採用。

| 項目 | 仕様 |
|------|------|
| 切替方法 | ステータスバーの「🚩 フラグ」ボタンをタップ |
| ON時の動作 | マスタップでフラグの設置/解除（マスは開かない） |
| OFF時の動作 | マスタップで通常の開封 |
| 視覚フィードバック | ON時: ボタンがオレンジに変化、盤面枠が黄色光 |
| PC対応 | 右クリック（onContextMenu）でもフラグ設置可 |
| フラグ数制限 | なし |

### 2-3. ライフシステム

| 項目 | 値 |
|------|-----|
| 初期ライフ | 3 |
| 犬踏み時 | -1（噛みつき演出800ms後に減算） |
| ライフ0 | ゲームオーバー（全マスオープンして終了） |
| ステージ間 | 各ステージ開始時に3にリセット |
| 表示形式 | ❤️ ♥♥♥ / ♥♥♡ / ♥♡♡ |

### 2-4. ヒントシステム

- ステージ開始時、**安全で猫が周囲にいるマス**を1つ自動選出
- 選出条件: `type==="empty" && dogCount===0 && catCount>0`
- パーティクル収束 → フラッシュ → バッジ表示の750ms演出で誘導
- タップでスキップ可能

### 2-5. クリア条件と敗北条件

| 条件 | トリガー | 結果 |
|------|---------|------|
| ステージクリア | そのステージの猫を全匹保護 | スコア表示 → 次ステージ or 出会い画面 |
| パーフェクトクリア | ライフ3を維持してクリア | 追加ボーナス + 特別演出 |
| ゲームオーバー | ライフが0になる | 全マスオープン → リトライ or タイトルへ |

### 3-2. 難易度カーブ

- ステージ1→2: グリッド拡大（64→81マス）、犬密度ほぼ同等
- ステージ2→3: グリッド拡大（81→100マス）、犬密度が顕著に上昇
- ステージ3→4: グリッドサイズ同じ、犬が3匹追加（密度15%へ）
- 後半ステージほどスコア倍率が高い → ハイスコア狙いの動機

### 3-3. 周回による難易度上昇

周回ごとに全ステージの犬が+1匹。上限は盤面の30%。

| 1周目 | 6 | 15 | 15.0% |
| 2周目 | 7 | 16 | 16.0% |
| 5周目 | 10 | 19 | 19.0% |
| 10周目 | 15 | 24 | 24.0% |

## 5. スキルシステム

### 5-1. お供選択

- タイトル画面の「おともの ねこ」カードで選択
- ◀ ▶ ボタンでアンロック済みの猫を循環
- 選んだ猫のスキルがそのプレイ中ずっと使用可能

### 5-2. ゲージ

| アクション | ゲージ加算 |
|-----------|----------|
| 空白/数字マス手動開封 | +5 |
| フラッドフィル追加マス | +1/マス |
| 猫を保護 | +25 |
| 犬を踏む | +0 |

最大100。満タンで発動可能。回数制限なし。

### 5-3. スキル分類

**即時効果型:** heal, pawhit, line, rush
**バフ型:** lucky（1回防御）, barrier（3秒防御）
**情報型:** mark（猫位置6秒表示）, peek（犬3秒表示）
**プレイヤー選択型:** cross（中心選択→十字9マス）, foresee（3マスの中身を確認）

詳細は `skill-logic.md` を参照。

## 6. スコアシステム

### 6-1. リアルタイム加算

猫保護+200、マス開封+5、数字マス手動開封+10

### 6-2. クリア時ボーナス

クリアボーナス+500、ライフボーナス（残×200）、パーフェクト+1000、高速クリア（<30s +1000 / <60s +500）、フラグマスター+500、無傷クリア+500、ノースキル+300

### 6-3. ステージ倍率

全ボーナス合計 × ステージ倍率（1.0〜2.0）= 最終スコア

詳細は `scoring-detail.md` を参照。

## 7. 周回システム

### 7-1. ゲームフロー

```
タイトル画面 → ステージ1〜4を順にプレイ → はいきょクリア
  → 「仲間との出会い」画面 → 新しい猫がアンロック
  → タイトルへ戻る（周回+1、難易度上昇）
```

### 7-2. lapCats（周回猫記録）

- 各ステージクリア時に保護した猫キーを `lapCats` に追加
- はいきょクリア後、`lapCats` の中から未アンロック猫を抽選
- 同じ猫を複数回保護していると候補枠が増え当選確率が上がる

### 7-3. リトライ時のデータ保持

| アクション | lapCats | 周回数 |
|-----------|---------|--------|
| リトライ | 前ステージまでの分を維持 | 変わらない |
| タイトルへ | 全リセット | 変わらない |
| 出会い画面→タイトル | リセット | +1 |

### 7-4. 仲間との出会い画面

**フェーズ:**

| フェーズ | 内容 | トリガー |
|---------|------|---------|
| idle | 「のぞいてみる」ボタン表示 | 画面表示時 |
| rustling | 草むらが激しく揺れる + ❓表示 | ボタン押下 |
| emerging | 猫がせり上がる + 光の輪 | 1.8秒後 |
| revealed | 猫名表示 + スキル情報 + 「なかまにする！」 | 2.8秒後 |
| empty | 「新しい出会いはなかった…」 | 候補なし |
| complete | 「全てのおともが集結！」+ 全猫並び | 全コンプ時 |

## 8. 用語集

| 用語 | 意味 |
|------|------|
| お供 | プレイヤーが選んだ猫。そのスキルがゲーム中使える |
| 保護 | 猫マスを開けること。「捕獲」や「攻撃」ではなく「保護」 |
| 噛みつき | 犬マスを開けたときのダメージ演出 |
| フラッドフィル | 安全な空マスが連鎖的に開く仕組み |
| 周回 | ステージ1〜4を通しでクリアすること |
| 出会い | はいきょクリア後に新しい仲間猫をアンロックするイベント |
| ラップ猫（lapCats） | 1周回で保護した全猫の記録 |

### 非暴力用語ルール

ゲーム内では以下の暴力的用語を避ける。

| NG | OK |
|----|-----|
| 攻撃 | 保護、サーチ |
| 殺す、倒す | 見つける、保護する |
| 爆破 | ラッシュ、サーチ |
| 武器 | スキル |
| 敵 | 野良犬 |
| 戦闘 | 探索 |
