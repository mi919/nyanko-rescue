# プロジェクト概要

パズルゲームの開発を行うプロジェクト。

マインスイーパーベースの猫救出パズルゲーム。犬（危険）を避けて猫（お宝）を全匹保護するとステージクリア。周回ごとに仲間の猫が増え、お供として連れていける。

## プロジェクト構造

Vite + React + Dockerプロジェクトへの再構築が目標。

プロトタイプは sample/ に配置。
単一JSXファイル（`nyanko-rescue.jsx`、2850行、469KB）のプロトタイプを参考資料とする。

## 開発ルール

### モバイルファースト
- 長押し非依存（フラグはトグルボタン方式）
- トーストは`position: fixed`（盤面レイアウトを動かさない）
- 演出は`pointerEvents: none`（誤タップ防止）
- シェイクは盤面コンテナのみ（画面酔い防止）
- セルサイズ: 38px固定

### テンポ重視
- ヒント演出: 750ms
- 猫保護演出: 700ms
- 犬噛みつき演出: 1.1s（最も長い、インパクト重視）
- ゲーム操作をブロックする演出は最小限に

### 技術方針
- React 18+、フックベース
- Dockerコンテナー
- 外部ライブラリ最小限
- `localStorage`による永続化
- CSS keyframes主体のアニメーション

### 用語
- 暴力的表現を避ける: 「攻撃」→「保護/サーチ」、「爆破」→「ラッシュ」、「敵」→「野良犬」

## ゲームの要点

- 数字は`犬数/猫数`の2色表示（赤=犬、緑=猫）
- ライフ3、犬を踏むと-1
- 全猫保護でクリア（安全マス全開封は不要）
- 10種類の猫 × 10種類のスキル（1対1対応）
- お供選択式（タイトル画面で1匹選ぶ）
- ゲージ制（最大100、マス開封+5、猫保護+25）
- 4ステージ制（にわ→こうえん→しょうてんがい→はいきょ）
- 周回で犬+1/ステージ（上限30%）
- はいきょクリア→出会い画面→仲間アンロック
- リトライ時は前ステージのlapCats保持
